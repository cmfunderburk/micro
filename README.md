# microecon

Research-first agent-based microeconomics platform that gives canonical theoretical microeconomics computational form.

## Core Insight

**Institutional visibility**: making economic institutions (bargaining protocols, information structures, search mechanisms) explicit, configurable, and comparable.

## Status

Early development. See [VISION.md](VISION.md) for project identity and methodology.

## Theoretical Grounding

All behavioral rules and mechanisms require formal justification from canonical sources (Kreps, Osborne & Rubinstein, MWG). See [theoretical-foundations.md](theoretical-foundations.md) for textbook mappings.
