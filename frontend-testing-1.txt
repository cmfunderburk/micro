  How to Run

  You'll need two terminals:

  Terminal 1 - Backend (port 8000):
  uv run python -m backend

  Terminal 2 - Frontend (port 5173):
  cd frontend && npm run dev

  Then open http://localhost:5173 in your browser.

  What to Test

  1. Basic simulation - Play/pause/step/reset, speed slider
  2. Grid interaction - Hover agents for tooltips, click to select
  3. Overlays - Toggle trails, perception radius, trade connections in sidebar
  4. Charts - Welfare and trade count graphs update in real-time
  5. Trade inspection - Click a trade in "Trade History" panel to see Edgeworth box
  6. Network panel - Click the network icon to see trade network graph with force/circular layouts
  7. Configuration - Click settings icon to change n_agents, grid_size, protocols, etc.
  8. Export - Download button offers PNG/CSV/JSON export

  The connection indicator in the header should turn green when the backend is connected.
